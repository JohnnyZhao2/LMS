## 📋 完整清理脚本

创建一个清理脚本，可以一次性清理所有缓存文件：

```bash

# 清理脚本：cleanup.sh

# 1. 清理 Python 缓存

find . -type d -name "__pycache__" -exec rm -r {} + 2>/dev/null
find . -name "*.pyc" -delete
find . -name "*.pyo" -delete
echo "✅ Python 缓存清理完成"

# 2. 清理 .DS_Store
echo "清理 .DS_Store 文件..."
find . -name ".DS_Store" -type f -delete
echo "✅ .DS_Store 清理完成"

# 3. 清理 Hypothesis 缓存
echo "清理 Hypothesis 缓存..."
rm -rf lms_backend/.hypothesis/
echo "✅ Hypothesis 缓存清理完成"

# 4. 清理临时文档（可选，注释掉以保留）
echo "清理临时文档..."
cd lms_backend

echo "✅ 临时文档清理完成"
```

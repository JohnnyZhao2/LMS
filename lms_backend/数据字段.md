## 数据字段定义

### 用户信息字段
- 用户ID (userId)
- 用户名 (username)
- 姓名 (realName)
- 工号 (employeeId)
- 角色 (role): 管理员/团队经理/室经理/导师/学员
- 所属室 (department): 一室/二室/无
- 是否为导师 (isMentor): 布尔值
- 账号状态 (status): 正常/停用
- 创建时间 (createdAt)
- 最后登录时间 (lastLoginAt)

### 知识条目字段
- 知识ID (knowledgeId)
- 标题 (title)
- 知识分类 (category): 双云告警/网络告警/数据库告警/应急场景/规章制度
- 子分类 (subCategory): 具体系统名称或场景名称
- 内容 (content): Markdown格式
- 标签 (tags): 数组，用于搜索
- 创建人 (createdBy)
- 创建时间 (createdAt)
- 最后修改人 (lastModifiedBy)
- 最后修改时间 (lastModifiedAt)
- 状态 (status): 有效/失效
- 阅读次数 (viewCount)
- 收藏次数 (favoriteCount)

### 题目字段
- 题目ID (questionId)
- 题目类型 (type): 单选/多选/简答/实操
- 题目内容 (content)
- 选项 (options): 数组，仅选择题有
- 正确答案 (correctAnswer)
- 分值 (score)
- 知识分类 (category): 双云/网络/数据库/应急/制度
- 关联知识ID (relatedKnowledgeIds): 数组
- 难度 (difficulty): 简单/中等/困难
- 创建人 (createdBy)
- 创建时间 (createdAt)
- 状态 (status): 有效/失效
- 使用次数 (usageCount)

### 任务字段
- 任务ID (taskId)
- 任务名称 (taskName)
- 任务类型 (taskType): 学习/练习/考试
- 任务描述 (description)
- 创建人 (createdBy): 导师或室经理的用户ID
- 创建人姓名 (creatorName)
- 创建时间 (createdAt)
- 开始时间 (startTime)
- 截止时间 (deadline)
- 时间限制 (timeLimit): 分钟数，0表示无限制
- 及格分数 (passingScore)
- 总分 (totalScore)
- 关联知识IDs (knowledgeIds): 数组，学习任务必填
- 题目IDs (questionIds): 数组，练习和考试必填
- 目标人员 (targetUsers): 数组，指定学员ID或"全部"
- 目标室 (targetDepartment): 一室/二室/全部
- 任务状态 (status): 草稿/已发布/已结束
- 是否允许查看答案 (allowViewAnswer): 布尔值
- 是否允许重做 (allowRetake): 布尔值

### 任务提交字段
- 提交ID (submissionId)
- 任务ID (taskId)
- 学员ID (learnerId)
- 学员姓名 (learnerName)
- 开始时间 (startTime)
- 提交时间 (submitTime)
- 答案 (answers): 对象数组，每个题目的答案
- 客观题得分 (objectiveScore): 自动评分
- 主观题得分 (subjectiveScore): 人工评分
- 总得分 (totalScore)
- 是否及格 (isPassed): 布尔值
- 评阅状态 (gradingStatus): 待评阅/已评阅
- 评阅人 (gradedBy)
- 评阅时间 (gradedAt)
- 评语 (comments)

### 现场抽查字段
- 抽查ID (quizId)
- 抽查人 (quizzerBy): 导师或室经理ID
- 抽查人姓名 (quizzerName)
- 被抽查人 (quizzeeId)
- 被抽查人姓名 (quizzeeName)
- 抽查时间 (quizTime)
- 知识主题 (knowledgeTopic): 选择的知识分类
- 评分 (score)
- 评语 (comments)
- 抽查地点 (location): 可选

### 学习进度字段
- 进度ID (progressId)
- 学员ID (learnerId)
- 知识ID (knowledgeId)
- 阅读状态 (readStatus): 未读/已读/收藏
- 阅读时长 (readDuration): 秒数
- 最后阅读时间 (lastReadAt)

### 收藏字段
- 收藏ID (favoriteId)
- 学员ID (learnerId)
- 知识ID (knowledgeId)
- 收藏时间 (favoritedAt)
